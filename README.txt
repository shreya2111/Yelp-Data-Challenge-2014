COGPARSER

1. Unzip all the files in a folder
2. Run 'input_to_cogparser.py'. This script uses the file 'business_review.json' and generates the file 'const_sentences.json'. The generated file gets saved in the directory 'dataset/'
3. Run 'Main.py'. This script uses the file 'const_sentences.json' as an input, and generates the files 'conceptnet_vector.json' and 'cogparsed_reviews.json'.
